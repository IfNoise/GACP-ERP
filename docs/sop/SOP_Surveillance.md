## 1. Purpose

Обеспечение видеонаблюдения и физической охраны объекта в соответствии с требованиями безопасности.

## 2. Scope

Все производственные, складские и лабораторные зоны объекта.

## 3. Responsibilities

- Security Manager: управление системой безопасности
- Security Guards: физическая охрана и мониторинг
- IT Admin: техническое обслуживание системы видеонаблюдения

## 4. Procedure

1. Круглосуточное видеонаблюдение критических зон
2. Контроль периметра и точек доступа
3. Патрулирование территории
4. Мониторинг сигнализации
5. Реагирование на инциденты безопасности
6. Архивирование видеозаписей

## 5. References

- URS-SEC-002, FS-SEC-002
- Требования лицензирования, нормы безопасности

## 6. Notes

- Видеозаписи интегрированы с ERP модулем Security Management

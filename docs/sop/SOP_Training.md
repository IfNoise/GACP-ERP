# SOP: Training and Competency Management

## 1. Purpose

Обеспечить систематическое обучение, оценку компетенций и аттестацию персонала в соответствии с требованиями GACP для гарантии качества работы и соответствия нормативным стандартам в производстве медицинского каннабиса.

## 2. Scope

Данная процедура охватывает:

- Всех сотрудников, работающих с продукцией и процессами
- Новых сотрудников (onboarding training)
- Периодическое обучение действующего персонала
- Специализированные тренинги по ролям
- Подрядчиков и временных работников
- Переподготовку при изменении процедур

## 3. Responsibilities

| Роль                             | Ответственность                                     |
| -------------------------------- | --------------------------------------------------- |
| **Training Manager**             | Общее управление программой обучения                |
| **HR Manager**                   | Координация обучения с жизненным циклом сотрудников |
| **Quality Manager**              | Одобрение учебных программ и стандартов             |
| **Department Heads**             | Определение потребностей в обучении                 |
| **Subject Matter Experts (SME)** | Разработка специализированного контента             |
| **Line Supervisors**             | Контроль практического применения знаний            |
| **Employees**                    | Активное участие в обучении и соблюдение требований |

## 4. Training Framework

### 4.1 Competency Matrix

#### 4.1.1 Core Competencies (для всех сотрудников)

| Компетенция               | Описание                               | Уровень |
| ------------------------- | -------------------------------------- | ------- |
| **GACP Knowledge**        | Основы надлежащей практики выращивания | Basic   |
| **Data Integrity**        | Принципы ALCOA+                        | Basic   |
| **Safety & Hygiene**      | Правила безопасности и гигиены         | Basic   |
| **Quality Awareness**     | Понимание системы качества             | Basic   |
| **Regulatory Compliance** | Нормативные требования                 | Basic   |

#### 4.1.2 Role-Specific Competencies

```yaml
Cultivation_Team:
  - Plant biology and cultivation techniques
  - Pest and disease management
  - Environmental controls
  - Harvesting procedures
  - Record keeping

Quality_Team:
  - Laboratory techniques
  - Analytical methods
  - Statistical analysis
  - Investigation procedures
  - Documentation practices

Production_Team:
  - Manufacturing processes
  - Equipment operation
  - Batch records
  - Material handling
  - Packaging procedures

Management_Team:
  - Leadership and supervision
  - Risk management
  - Change control
  - Performance management
  - Regulatory knowledge
```

### 4.2 Training Categories

| Category                 | Purpose                            | Frequency   |
| ------------------------ | ---------------------------------- | ----------- |
| **Initial Training**     | New employee onboarding            | Once        |
| **Annual Refresher**     | Knowledge update and reinforcement | Yearly      |
| **Specialized Training** | Role-specific skills               | As needed   |
| **Regulatory Update**    | Changes in regulations             | As required |
| **Emergency Training**   | Critical procedures                | As needed   |

## 5. Training Process

### 5.1 Training Needs Assessment

1. **Job Analysis**

   - Review position requirements
   - Identify knowledge gaps
   - Determine skill levels needed
   - Map to competency matrix

2. **Individual Assessment**

   - Current knowledge evaluation
   - Experience assessment
   - Performance review analysis
   - Development needs identification

3. **Organizational Requirements**
   - Regulatory mandates
   - Business objectives
   - Process changes
   - Technology updates

### 5.2 Training Plan Development

1. **Curriculum Design**

   - Learning objectives definition
   - Content development
   - Training methods selection
   - Assessment criteria establishment

2. **Resource Planning**

   - Trainer assignment
   - Material preparation
   - Facility booking
   - Technology requirements

3. **Schedule Creation**
   - Training calendar
   - Individual schedules
   - Deadline management
   - Conflict resolution

### 5.3 Training Delivery

#### 5.3.1 Training Methods

| Method                  | Best For              | Advantages                      |
| ----------------------- | --------------------- | ------------------------------- |
| **Classroom Training**  | Theoretical knowledge | Interactive, immediate feedback |
| **On-the-Job Training** | Practical skills      | Real-world application          |
| **E-Learning**          | Standard procedures   | Self-paced, consistent          |
| **Simulation**          | Critical procedures   | Safe practice environment       |
| **Mentoring**           | Complex skills        | Personalized guidance           |

#### 5.3.2 Training Documentation

Each training session must include:

- **Training Record Form**

  - Date, time, duration
  - Trainer and participants
  - Topics covered
  - Materials used

- **Attendance Verification**

  - Electronic signatures
  - Photo documentation
  - Badge scanning

- **Content Documentation**
  - Presentation materials
  - Handouts and references
  - Video recordings (if applicable)

### 5.4 Competency Assessment

#### 5.4.1 Assessment Methods

1. **Written Examinations**

   - Multiple choice questions
   - Case study analysis
   - Scenario-based problems
   - Regulatory knowledge tests

2. **Practical Demonstrations**

   - Skill demonstrations
   - Equipment operation
   - Procedure execution
   - Problem-solving exercises

3. **Performance Observations**
   - On-the-job evaluation
   - Supervisor assessments
   - Peer reviews
   - Self-assessments

#### 5.4.2 Passing Criteria

| Assessment Type        | Passing Score           | Retake Policy            |
| ---------------------- | ----------------------- | ------------------------ |
| **Written Exam**       | 80% minimum             | Max 2 retakes            |
| **Practical Demo**     | Satisfactory completion | Immediate retake allowed |
| **Performance Review** | Meets expectations      | Continuous improvement   |

### 5.5 Certification and Records

1. **Training Certificates**

   - Completion certificates
   - Competency certifications
   - Expiration dates
   - Renewal requirements

2. **Training Records**
   - Individual training history
   - Competency status
   - Assessment results
   - Performance tracking

## 7. Training Records Management

### 7.1 Documentation Requirements

#### 7.1.1 Individual Training Files

```yaml
Employee_Training_Record:
  personal_info:
    employee_id: "unique_identifier"
    name: "full_name"
    position: "job_title"
    department: "department_name"
    start_date: "employment_start"

  training_history:
    - course_name: "training_title"
      date_completed: "completion_date"
      trainer: "instructor_name"
      duration: "hours"
      score: "assessment_result"
      certificate_number: "cert_id"
      expiration_date: "renewal_due"

  competency_status:
    current_certifications: []
    expired_certifications: []
    pending_training: []
    next_due_date: "upcoming_deadline"
```

### 7.2 Record Retention

| Record Type            | Retention Period                 | Storage Location    |
| ---------------------- | -------------------------------- | ------------------- |
| **Training Records**   | Duration of employment + 3 years | ERP Training Module |
| **Certificates**       | Life of certificate + 1 year     | Digital archive     |
| **Assessment Results** | 5 years                          | Secure database     |
| **Training Materials** | Life of procedure + 2 years      | Document control    |

### 7.3 Access and Confidentiality

- **Authorized Access**: HR, Training Manager, Direct Supervisor
- **Privacy Protection**: Personal information security
- **Audit Trail**: All access logged
- **Data Integrity**: Regular backup and verification

#### Level 4: Results
